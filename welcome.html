apollo-kotlin-normalized-cache Help

Welcome

This is Apollo Kotlin's Normalized Cache.

For an introduction please read the Normalized Cache documentation.

Note: the Normalized Cache used to be part of the Apollo Kotlin main repository. It is now hosted in this dedicated repository and published at its own cadence and versioning scheme.

Use in your project

During the alpha phase, the API is still subject to change, although we will try to make changes in non-breaking ways. For now it is recommended to experiment with this library in non-critical projects/modules, or behind a feature flag.

Add the dependencies to your project.

// build.gradle.kts dependencies { // For the memory cache implementation("com.apollographql.cache:normalized-cache:1.0.0-alpha.2") // For the SQL cache implementation("com.apollographql.cache:normalized-cache-sqlite:1.0.0-alpha.2") }

If you were using the classic Normalized Cache before, please consult the migration guide.
